Frin CSS

A tiny, pure-CSS component framework. Mobile-first, accessible by default, and themeable with CSS custom properties. No build step required.

Browse components GitHub

Why Frin CSS

🎨 Themeable

Every color, space and radius is a CSS variable — re-skin in seconds.

🌗 Dark mode

Built-in light/dark themes via a single data-theme attribute.

♿ Accessible

Visible focus rings, ARIA patterns and reduced-motion support.

📱 Mobile-first

Responsive grid and components by default, no media-query soup.

🪶 Tiny

~34 KB minified, zero dependencies, zero build step.

🔧 Optional JS

Components render without JS; enhancements only add interactivity.

Theming

Override CSS variables in your own stylesheet — no build step, no editing core files. Toggle the 🌙 button above for dark mode.

:root {
  --frin-color-primary: #7c3aed;
  --frin-radius-md: 4px;
  --frin-font-family: "Inter", sans-serif;
}
[data-theme="dark"] {
  --frin-color-bg: #0d1424;
}
