Frin CSS

A tiny, pure-CSS component framework. Mobile-first, accessible by default, and themeable with CSS custom properties. No build step required.

Browse components GitHub

Components

Buttons

Variants: --primary, --secondary, --outline, --ghost. Sizes: --sm, --lg. Add is-disabled or [disabled].

<button class="frin-btn frin-btn--primary">Primary</button>
<button class="frin-btn frin-btn--outline frin-btn--lg">Large</button>
<button class="frin-btn frin-btn--primary" disabled>Disabled</button>

Modal

Theming

Override CSS variables in your own stylesheet — no build step, no editing core files. Toggle the 🌙 button above for dark mode.

:root {
  --frin-color-primary: #7c3aed;
  --frin-radius-md: 4px;
  --frin-font-family: "Inter", sans-serif;
}
[data-theme="dark"] {
  --frin-color-bg: #0d1424;
}
